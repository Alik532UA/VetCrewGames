# Vet Crew Games

Набір розвиваючих ігор про тварин.

## Проекти

- **Кого більше?** (Population game) — гра, де потрібно розставити тварин за чисельністю їхньої популяції.

## Технології

- Svelte 5
- SvelteKit
- TypeScript
- Lucide Icons
- GitHub Pages

## Розробка

```bash
npm install
npm run dev
```

## Деплой

Автоматично через GitHub Actions при пуші в гілку `main`.
